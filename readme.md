# попытка построить фичестор
[датасет](https://www.kaggle.com/datasets/mkechinov/ecommerce-behavior-data-from-multi-category-store)

- dataset_processing - процессинг датасета, в частности
    - разбивка на подневные файлы и сохранение в паркеты
    - чеки для сравнения сырых csv и кликхауса
    - описание загрузки сверочных данных в кх
- data_producer - скрипт, эмулирующий отправку данных с бэкенда. берет ивенты из паркетников и пуляет их в кафку
